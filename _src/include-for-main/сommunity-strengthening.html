Зміцнюємо громади для порятунку сімей з дітьми

Навчаємо та підвищуємо компетентності фахівців, волонтерів для надання якісної підтрмики сімей з дітьми

  • Якісне навчання для тих, хто працює чи хоче працювати з сім'ями з дітьми (волонтери, соціальні робітники)

  • Можливості для постійного підвищення кваліфікації ФСРів, соціальних працівників, менеджерів соціальної роботи

  • Професійний супровід під час надання соціальної підтримки з сім’ями з дітьми
